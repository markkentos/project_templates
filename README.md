# Anime Shelf

Django-веб-приложение для интернет-магазина аниме атрибутики. Проект сделан под семестровую работу: есть БД, заполнение демо-данными, пользовательский интерфейс, математическая модель и демонстрация паттернов проектирования.

## Запуск

```powershell
python manage.py migrate
python manage.py seed_store
python manage.py runserver
```

После запуска:

- каталог: http://127.0.0.1:8000/
- демонстрация паттернов: http://127.0.0.1:8000/patterns/
- дерево каталога: http://127.0.0.1:8000/tree/
- админка: http://127.0.0.1:8000/admin/

## Что реализовано

- каталог товаров, поиск, категории, карточки товаров;
- корзина, пересчет скидок и доставки через стратегии;
- оформление заказа через шаблонный метод процесса;
- машина состояний заказа: создан, оплачен, упакован, отправлен, доставлен, отменен;
- журнал процесса через наблюдателя;
- рекомендации через адаптер внешнего модуля трендов;
- клиентский каталог через заместитель;
- иерархия каталога через компоновщик и итератор;
- генерация товаров через абстрактную фабрику, декоратор и шаблонный метод;
- линейная математическая модель прогноза спроса по недельным точкам продаж.
