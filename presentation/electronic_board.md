# Электронная доска для демонстрации

## 1. Цель

Интернет-магазин аниме атрибутики с каталогом, корзиной, заказами, прогнозом спроса и демонстрацией паттернов.

## 2. Основные процессы

- Покупка товара: каталог -> корзина -> стратегии -> заказ -> события -> журнал.
- Наполнение каталога: seed-команда -> фабрики -> декораторы -> товары -> точки продаж -> прогноз.

## 3. Паттерны

- Strategy: скидки и доставка.
- Template Method: оформление заказа и генерация товаров.
- Abstract Factory: типы мерча.
- Decorator: limited edition и gift wrap.
- State: статусы заказа.
- Singleton: настройки, EventBus, реестры стратегий.
- Adapter: модуль трендов.
- Observer: журнал и уведомления.
- Command: действия пользователя.
- Proxy: клиентский каталог.
- Composite + Iterator: дерево категорий.

## 4. Демо-маршрут

1. Открыть `/`.
2. Найти товар `Gojo`.
3. Открыть карточку и показать прогноз спроса.
4. Добавить товар в корзину.
5. Переключить скидку и доставку.
6. Оформить заказ.
7. На странице заказа менять состояния.
8. Открыть `/patterns/` и `/tree/`.
